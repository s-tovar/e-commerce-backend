## Description 

This E-commerce Backend is an Express.js API application designed to meet the needs of modern internet retail companies by providing powerful, scalable, and eddicient backend. This application offers users the ability to leverage the latest technologies and competes effectively with other e-commerce businesses. 

## Installation 
***Requirements***

[MySQL2](https://www.npmjs.com/package/mysql2) | [Sequelize](https://www.npmjs.com/package/sequelize) | [dotenv](https://www.npmjs.com/package/dotenv)

***Once Installed***
1. Clone the Repository on to your machine.
2. Open the terminal and ensure you are in the right file path.
3. Run the command ```npm install``` to download the packages.
4. Then run the command ```node server.js``` to run the software.

## Usage 

1. Run MySQL by running the command ```mysql -u root -p``` and type the user and password located in the .env file.
2. Run the following commands ```source db/schema.sql``` and after ```exit```
3. Run the command ```npm run seed```
4. Then run the command ```node server.js``` to run the software.
5. Test code on application Insomnia.
